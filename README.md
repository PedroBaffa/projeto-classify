# Classify - Sistema de Escalas Institucionais

O projeto é um front-end em React/TypeScript (usando Vite) para um sistema web, focado em reutilização de componentes. A arquitetura usa React Router para dividir o site em dois grupos: 1) Landing Page (páginas públicas) e 2) Autenticação (páginas de login/cadastro). O "Grupo 1" usa um Layout Universal (Layout.tsx) que inclui o Header (fixo, com "glassmorphism"), o Footer (fixo em baixo) e uma animação de fundo (@keyframes). O "Grupo 2" tem layouts limpos (2 colunas para Login/Cadastro e centralizado para Alterar Senha). Os componentes (como o Button.tsx) são "inteligentes", usando NavLink para mostrar o estado ativo (botão laranja) automaticamente. O TelaAlterarSenha é um "Wizard" que usa useState para controlar as etapas (Etapa 1: Senha, Etapa 2: Preferências). A estilização é feita com CSS Modules para evitar conflitos e um global.css (tema claro).
